# delivery-app
## Getting Started

First, run the development server:

Create .env files on backend and frontend, and fill them like in .env.example file

for backend:
```bash
cd backend
npm install
npm run dev
# or
yarn dev
# or
pnpm dev
```

for frontend:
```bash
cd frontend
npm install
npm run dev
# or
yarn dev
# or
pnpm dev
```
